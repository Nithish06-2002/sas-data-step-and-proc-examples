# sas-data-step-and-proc-examples
Real-time clinical data examples using SAS demonstrating DATA step, SET, KEEP, DROP, RENAME, WHERE, FIRSTOBS, OBS, MERGE, and PROC PRINT for beginners and interview preparation.

# Clinical Data Processing using SAS

This project demonstrates basic clinical data handling using SAS.
It includes real-world examples of Demographics (DM) and Laboratory (LB) datasets.

## Concepts Covered
- DATA step
- SET statement
- KEEP / DROP / RENAME
- WHERE filtering
- FIRSTOBS / OBS
- MERGE and JOIN logic
- Derived variables
- PROC PRINT

## Use Case
Suitable for:
- Clinical SAS beginners
- Interview preparation
- Academic learning
